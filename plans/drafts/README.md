# Draft Plans And Specs

Use this folder for branch-local planning drafts that should not claim stable
`SPEC-*`, `PLAN-*`, or `IMPL-*` IDs yet.

Drafts are useful when multiple worktrees or agents are preparing planning docs
in parallel. Promote a draft only after rebasing onto the branch it will merge
into, then assign the next stable ID with the current repository state.

Draft files should use descriptive names such as:

```text
SPEC-DRAFT-docs-ops-command-surface-and-draft-promotion.md
PLAN-DRAFT-docs-ops-command-surface-and-draft-promotion.md
```
